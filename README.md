## Deskripsi Aplikasi
PesantrenKita adalah aplikasi berbasis web yang dikembangkan dalam kegiatan
Project Based Learning (PBL).

Aplikasi ini bertujuan untuk membantu pengelolaan data pesantren secara terintegrasi,
meliputi data santri, kepegawaian, keuangan, inventaris, dan administrasi lainnya.
